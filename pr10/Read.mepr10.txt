# Решение линейных и квадратных уравнений

Эта программа позволяет решать линейные и квадратные уравнения вида B*x + C = 0 и A*x^2 + B*x + C = 0.

# Классы

Программа использует два класса: type5 и type6.

# Класс type5

Класс type5 предназначен для решения линейных уравнений вида B*x + C = 0.

# Конструктор

Конструктор класса type5 принимает два параметра: b1 и c1, которые являются коэффициентами уравнения B и C соответственно. Он инициализирует приватные поля класса B и C этими значениями.

# Методы

Класс type5 имеет два публичных метода: Get_answer() и show().

Метод Get_answer() находит корень уравнения по формуле x = -C/B и выводит его на экран.

Метод show() выводит уравнение на экран в виде B*x + C = 0.

# Класс type6

Класс type6 предназначен для решения квадратных уравнений вида A*x^2 + B*x + C = 0.

# Конструктор

Конструктор класса type6 принимает три параметра: a1, b1 и c1, которые являются коэффициентами уравнения A, B и C соответственно. Он инициализирует приватные поля класса A, B и C этими значениями.

# Методы

Класс type6 имеет два публичных метода: Get_answer() и show().

Метод Get_answer() находит корни уравнения по формуле x = (-B +- sqrt(D))/(2*A), где D = B^2 - 4*A*C - дискриминант. Если D < 0, то метод сообщает, что уравнение не имеет действительных корней. Если D = 0, то метод выводит один корень. Если D > 0, то метод выводит два корня.

Метод show() выводит уравнение на экран в виде A*x^2 + B*x + C = 0.
